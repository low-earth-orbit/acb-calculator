# ACB Calculator

## Wealthsimple

Calculates the adjusted cost base (ACB) for securities from Wealthsimple activity exports.

### Usage

```bash
python acb-wealthsimple.py
```

### Process

1. Select a CSV file from the `data/` folder (or provide a custom path)
2. Select a security symbol
3. Enter T3 box 21 and box 42 amounts
4. Get results: ACB and ACB per share
